What is Tomatenquark

What is Tomatenquark

Tomatenquark is a early stage project dedicated to the Cube 2: Sauerbraten game. Tomatenquark's primary objective is to bring back ongoing development of the game and it's amazing community. This effort has be initiated by providing binaries of the current svn. Changes to the current svn will be incorporated into Tomatenquark in the foreseeable future.

Obtaining Tomatenquark

Tomatenquark can easily be obtained using it's launcher. Please download the launcher for your platform below:

Macos Windows Ubuntu

Development

If you would like to contribute to the development of Tomatenquark (and thus the sauerbraten ecosystem) please note the following:

  • there is a overall direction that this project aims at - see "Sauerbraten fork - Tomatenquark"
  • the project is in very early phases. Feedback is highly appreciated. You can talk to me on Discord.
  • everything is undocumented right now. This is very much how sauerbraten deals with things, and very much how Tomatenquark is not supposed to be. Be a little patient. Creating quality documentation takes time 🍷
